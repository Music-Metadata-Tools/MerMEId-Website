## The Project

MerMEId MeLODy is the new generation of [MerMEId](https://github.com/Edirom/MerMEId), an open-source metadata editor for music-related research data. The original MerMEId was developed at the Danish Centre for Music Editing and has been used to create published catalogues for composers including Carl Nielsen, Anton Bruckner, Giuseppe Tartini, and Frederick Delius.

MerMEId MeLODy shifts the underlying data format from MEI/XML to RDF/Turtle, placing music metadata within the Linked Open Data landscape. The editor is designed to produce FAIR-compliant data that can be integrated with other LOD resources and published via SPARQL endpoints or static file hosting.

The editor is browser-based and requires no installation — data is stored in standard Git repositories, making it accessible to researchers without a technical background while remaining fully transparent and extensible for developers.
