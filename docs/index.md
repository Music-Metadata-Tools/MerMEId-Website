# Welcome to MerMEId

The MerMEId community is the central location for active exchange on all questions relating to the open source software MerMEId (Metadata editor and repository for MEI data). It brings together all active users of the software, musicologists, librarians, metadata enthusiasts, and developers. Here, questions regarding further development, use of the editor, experiences with older MEI versions, and general questions about metadata capture are discussed.

For testing of the latest beta version visit [here](https://adwmainz.pages.gitlab.rlp.net/nfdi4culture/cdmd/mermeid/), to get started you can read a short [How To] .
